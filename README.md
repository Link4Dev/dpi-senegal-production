# 🇸🇳 DPI Sénégal - Production Platform

Digital Public Infrastructure for:
- National Registry
- Social Protection (PBMS)
- Agricultural Data Systems
- Payments Integration
- AI-driven policy support

## Stack
- Odoo 16 (Registry + Apps layer)
- Python FastAPI (services)
- PostgreSQL HA
- Kafka (event streaming)
- Redis (cache)
- Kubernetes (production)
- Keycloak (SSO)
- Superset (analytics)

## Core Modules
- Registry (individuals, groups, households)
- PBMS (social programs)
- Agriculture data layer
- Payments gateway
- Interoperability APIs
